A custom lint rule that checks for the use of GoRouter.of(context).go() instead of context.go().

This lint rule is part of the go_router_linter plugin, which provides a set of custom lints for the Go Router library. The purpose of this lint is to encourage the use of the more concise and idiomatic context.go() method instead of the longer GoRouter.of(context).go() form.
