ClassDeclarationExtension

ClassDeclarationExtension extension

Provides convenient inspection utilities on a ClassDeclaration to help custom lint rules determine class characteristics in Flutter code. Utility getters include checks for private Flutter State classes, abstract declarations, interface implementations, superclass extensions, generic type usage, and overall relationships to other classes.
